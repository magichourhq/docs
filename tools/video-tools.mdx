## Animation API

**What it does:** Transform static images into dynamic animated videos with smooth motion and camera effects. This API creates engaging stop-motion style animations, adding movement, depth, and visual interest. You can optionally provide an image as a starting point and add audio for synchronized effects.

**Use cases:**

- Music videos
- Social media content creation
- B-roll video content
- Creative video content for apps
- Animated profile pictures and avatars

**API Endpoint:** `POST /v1/animation`

## Face Swap Video API

**What it does:** Replace faces in videos with frame-by-frame precision and temporal consistency. This API swaps faces throughout video sequences while maintaining natural expressions, head movements, and lighting for realistic results.

**Use cases:**

- Entertainment and viral content creation
- Privacy protection in video content
- Character replacement in film and media
- Personalized video experiences
- Creative video editing applications

**API Endpoint:** `POST /v1/face-swap`

## Lip Sync API

**What it does:** Synchronize lip movements in videos with new audio tracks using advanced AI motion analysis. This API creates realistic lip-sync animation that matches speech patterns, timing, and mouth movements for natural-looking results.

**Use cases:**

- Dubbing and localization for global content
- Personalized video messages and greetings
- Educational and training video production
- Entertainment and social media content
- Accessibility features for video content

**API Endpoint:** `POST /v1/lip-sync`

**What it does:** Automatically generate accurate subtitles and captions for video content using advanced speech recognition and natural language processing. This API creates properly timed, formatted subtitles in multiple languages with customizable styling.

**Use cases:**

- Accessibility compliance for video content
- Multi-language content localization
- Social media video optimization
- Educational and training video enhancement
- Automated video post-production workflows

**API Endpoint:** `POST /v1/auto-subtitle-generator`
