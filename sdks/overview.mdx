## Common Features

All Magic Hour SDKs include:

- **Type Safety**: Complete type definitions for all API endpoints and responses
- **File Upload**: Simplified file upload handling for images and videos
- **Async Support**: Modern async/await patterns for non-blocking operations
- **Error Handling**: Comprehensive error types and handling
- **Auto-retry**: Built-in retry logic for transient failures
- **Polling Utilities**: Helper functions to wait for job completion
- **Documentation**: Extensive examples and API documentation

## Getting Started

1. **Choose your SDK** based on your preferred programming language
2. **Install the SDK** using your language's package manager
3. **Get your API key** from the [Magic Hour Developer Hub](https://magichour.ai/developer?tab=api-keys)
4. **Follow the quickstart guide** for your chosen SDK
