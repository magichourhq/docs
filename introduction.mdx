**Key features:**

- **Video generation**: Create videos from text, images, or reference videos 
- **Video editing**: Swap faces, add subtitles, change styles, and animate videos
- **Image generation**: Produce images from prompts or reference images
- **Image editing & enhancement**: Edit, swap faces, remove backgrounds, and upscale images
- **REST API**: Simple HTTP requests with JSON responses  

Magic Hour manages the models, scaling, and infrastructure so you can focus on building your application.
