## High level diagram

The diagram shows a high level diagram of how webhook works.

```mermaid
sequenceDiagram
    participant app as Your Application
    participant mh as Magic Hour

    app->>mh: Create video or image
    mh->>mh: Generate Payload & Signature
    mh->>app: Send Webhook (Payload + Signature)

    app->>app: Validate Signature
    app->>mh: Return 2xx Response
```

## Setup webhook

<Steps>
<Step title="Visit Developer Hub">
Go to Magic Hour [Developer Hub](https://magichour.ai/developer), and click `Create Webhook`

![Webhook Table](./images/webhook-secret.png)

</Step>
<Step title="Configure webhook">
  
On the modal, input

- Your API endpoint to receive events
- Select the events you're interested in receiving.

![Create webhook modal](./images/create-webhook-modal.png)

Then click `Create Webhook`

</Step>
<Step title="Save webhook secret">

Copy and save the webhook secret.

![Webhook Secret](./images/webhook-secret.png)

<Note>This secret can be retrieved later.</Note>

</Step>

</Steps>
